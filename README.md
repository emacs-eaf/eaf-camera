Camera application for the [Emacs Application Framework](https://github.com/emacs-eaf/emacs-application-framework).

### Load application

```Elisp
(add-to-list 'load-path "~/.emacs.d/site-lisp/eaf-camera/")
(require 'eaf-camera)
```

### The keybinding of EAF Camera.

| Key   | Event   |
| :---- | :------ |
| `p` | take_photo |
